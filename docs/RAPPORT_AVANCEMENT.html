1. Contexte et Objectif

Article de reference : Prattico et al. (2025) — "Enhancing Power Quality and Reducing Costs in Hybrid AC/DC Microgrids via Fuzzy EMS" — Energies MDPI 18(22) 5985.

Objectif : Reproduire le modele complet du microgrid hybride AC/DC en Simscape Electrical comme base experimentale pour la these. L'approche retenue est une construction incrementale phase par phase, avec validation automatisee a chaque etape.

Approche incrementale
Chaque phase ajoute un composant et execute 12 tests. Aucune phase n'est validee tant que tous les tests ne passent pas.

6. Direction Proposee — AutoResearch

PROPOSITION

Decouverte Autonome d'EMS Optimaux via IA Agentique

Inspire par AutoResearch (Karpathy, Mars 2026), nous proposons d'utiliser un agent LLM pour explorer automatiquement l'espace des parametres EMS. L'agent lit le code FIS, propose des modifications, lance la simulation MATLAB, et garde les ameliorations.
