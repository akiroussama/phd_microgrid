# 99. Candidats secondaires

## Prediction-Free Coordinated Dispatch of Microgrid

- URL: <https://data.mendeley.com/datasets/8j2zdf6h5j/2>
- Interet:
  - utile comme **data de test** pour dispatch microgrid;
  - inclut des elements de type charge, PV, vent, prix et configuration;
- bon candidat si vous voulez rapidement un cas public oriente optimisation/dispatch.
- Pourquoi non prioritaire ici:
  - moins fort que Mesa Del Sol pour la validation electrique;
  - moins standard que SimBench pour le benchmark;
  - moins directement exploitable que NASA POWER pour reconstruire le site Prattico.
